# F7.1 — Confluence-Style Wiki Page

## Overview

Auto-generated wiki at `/wiki?project=NAME` rendering full vision.json as a navigable document with sections for principles, milestones, goals, features (with tasks/criteria/questions/decisions), ADRs, and changelog.

## Architecture

- **Endpoint**: `GET /wiki?project=NAME` → returns full HTML page
- **Data source**: Reads `.vision/vision.json` via `load_vision()`
- **Rendering**: Server-side HTML generation with dark theme matching dashboard
- **Navigation**: Anchor links to sections (principles, milestones, goals, features, ADRs)

## Features

- Goal cards with status badges and metrics
- Feature cards with task lists, acceptance criteria, Q&A sections
- Architecture Decision Records with rationale and alternatives
- Milestone progress bars
- Research/discovery doc links

Wiki renders correctly with all vision data. Linked from dashboard Vision section.
